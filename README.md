# 💸 Subscription Killer

A simple, fast web app to track all your subscriptions and see how much you're really spending monthly and yearly.

## Features

✅ **Add subscriptions** - Enter subscription name and monthly price  
✅ **Track total costs** - See your monthly and yearly spending at a glance  
✅ **Delete subscriptions** - Remove subscriptions with one click  
✅ **Local storage** - Your data is saved in your browser (no server needed)  
✅ **Lightweight** - Single HTML file, no dependencies  

## Live Demo

👉 **[Try it now](https://h78z2vzm87-glitch.github.io/Subscription-killer-/)**

## How to Use

1. Enter a subscription name (e.g., Netflix, Spotify, etc.)
2. Enter the monthly price in £
3. Click "Add Subscription"
4. See your total monthly and yearly costs
5. Click ❌ to remove any subscription

## Getting Started

1. Clone this repo
2. Open `index.html` in your browser
3. Start tracking your subscriptions!

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage API
